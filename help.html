Help & Support

Need help using Campus Connect? Follow the steps below:

Home Clubs Favorites Help
  1. Search for clubs using keywords or categories on the Home page.
  2. Click “View” or “Add to Favorites” to interact with clubs.
  3. Visit the “Favorites” page to see all saved clubs.
  4. Click “Clear Favorites” to remove all saved clubs.

If you still need assistance, contact us at support@campusconnect.com.
